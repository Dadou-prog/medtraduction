📋 Format du fichier JSON

Chaque entrée doit être un objet avec ces champs (tous optionnels sauf au moins un) :

[ { "fr": "douleur thoracique", "shi": "maumivu ya kifua", "gc": "maumivu ya kifua", "anj": "maumivu ya kifua", "cat": "Urgence" }, { "fr": "allergie", "shi": "aleji", "gc": "aleji", "anj": "aleji", "cat": "Antécédents" } ]

Les champs fr, shi, gc, anj et cat sont les seules clés lues. Les autres sont ignorées.
